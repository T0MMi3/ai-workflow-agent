# AI Workflow Agent

A Python-based AI workflow orchestration system that dynamically routes user requests to specialized automation pipelines using an LLM-powered agent.

The system demonstrates how AI can be integrated into enterprise workflow automation systems.

## Features

- AI agent that selects workflow tools
- FastAPI backend service
- Streamlit UI for interaction
- Structured JSON outputs
- Policy-grounded ticket classification
- Fallback handling when LLM is unavailable
- Workflow logging for auditability
